# React Todo App

## Description
This is a simple Todo List application built using React. Users can add, delete, and view tasks in ascending order.

## Features
- Add new tasks
- Delete tasks
- Tasks sorted in ascending order
- Input field clears after adding
- Simple and clean UI

## Technologies Used
- React JS
- JavaScript
- CSS

## How to Run

1. Install dependencies:
   npm install

2. Start the app:
   npm start

3. Open in browser:
   http://localhost:3000
